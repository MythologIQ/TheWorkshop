# MythologIQ Creation Lab

MythologIQ Creation Lab is a kids-first creative studio that helps children and curious beginners turn ideas into real projects with the help of AI.

This repository contains:
- Design documents and specs (in `docs/`)
- System and role prompts (in `prompts/`)
- UI flows and wireframes (in `ui/`)
- Application code (in `app/`)
- Operations and environment notes (in `ops/`)
- Roadmap and change history (in `roadmap/`)

Creation Lab is separate from Hearthlink. Hearthlink is a professional tool for orchestrating agents. Creation Lab is a teaching and exploration space designed for kids first.
